## 🚦 Hızlı Başlangıç

1. Depoyu klonlayın veya ZIP indirin.
2. `index.html` dosyasını tarayıcıda açın.
3. Sol panelden içeriğinizi girin, sağda canlı önizlemesini görün.
4. Yayınla sekmesinden ZIP indirin veya GitHub Pages’e yükleyin.

## 🛠️ Kullanılan Teknolojiler

- HTML5, CSS3, JavaScript (Vanilla)
- [JSZip](https://stuk.github.io/jszip/) – ZIP çıktısı
- [jsPDF](https://github.com/parallax/jsPDF) – PDF çıktısı
- Google Fonts
